Entity.java

Описание:
Кратко:
 Один из тех классов, которые делают MarteEngine столь удачным дополнением к Slick2D;
Entity - это игровая сущность. Человек, дерево, стена, и даже текст/элементы UI, если того пожелает разработчик - уровень абстракции Entity это позволяет. Этот класс очень тесно связан с World, и обладает встроенным обнаружением столкновений, механизмом поддержки анимаций, и другими полезными функциями.

Подробно:
 Entity - это первый класс в цепочке наследования сущностей
Пример кода можно посмотреть в тестовом проекте.

Поля
public static final String SOLID Первый из двух заранее предусмотренных типов
public static final String PLAYER Второй из двух заранее предусмотренных типов
public World world Мир(World), в котором живёт сущность
public String name Уникальный идентификатор сущности
public String name Уникальное имя сущности
public boolean centered Флаг, сигнализирующий о перемещении центра сущности из левого верхнего угла в настоящий центр
public int width Ширина сущности, и по-совместительству ширина хитбокса
public int height Высота сущности (и высота хитбокса соответственно)
public int height Ширина сущности, и по-совместительству ширина хитбокса
