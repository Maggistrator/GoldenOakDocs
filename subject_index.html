BasicGame
Простой самодостаточный класс-контейнер.
Состоит из 3-х основных методов - init, render и update. Эта архитектура подходит только для самых простых игр, а потому используется в основном новичками, или для очень небольших проектов.

StateBasedGame
Заготовка для полноценной игры основанной на сценах (классах состояний).
В нем может быть неограниченное число сцен, между которыми можно неограниченно перемещаться. Каждая сцена представляет собой аналог BasicGame, и называется BasicGameState.

BasicGameState
